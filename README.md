# DeepNut OSINT
[![Python Version](https://img.shields.io/badge/python-3.10+-3776AB?style=for-the-badge&logo=python&logoColor=white)](https://python.org)
[![WIP](https://img.shields.io/badge/status-work_in_progress-orange?style=for-the-badge)](https://github.com/realpnut/deepnut)
[![Maintained](https://img.shields.io/badge/maintained-yes-brightgreen?style=for-the-badge)](https://github.com/realpnut/deepnut)

tiny osint tool for the terminal. built for speed and clean looks.

    user search: find profiles with sherlock.

    ip tracker: fast ip lookup with maps.


clean, fast, and simple. 🥜🚀
    

## installation
```bash
git clone https://github.com/realpnut/deepnut.git && cd deepnut && chmod +x deepnut.sh && ./deepnut.sh
```
